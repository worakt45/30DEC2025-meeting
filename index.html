üöÄ Executive Summary

Great News: The project is running ahead of schedule!

We are moving the Pilot Testing phase with landlord users to January 7th (originally planned for March). This acceleration allows us to validate the MVP sooner and aim for a Go-to-Market launch in March.

üì± Product Spotlight: MVP Features

1. Front-End Dashboard (User Experience)

  • 5 User Roles: Tailored views for Admin, Landlord, Manager (Agents), and Tenants.
  • Admin Power: Full control over system indicators, transaction analytics, unit management, and finance balancing.
  • Service Integration: A dedicated tab for connecting with services (like HomePro), repairs, cleaning, and worker availability.
  • Customization: Supports Light/Dark mode, multi-language, and timezone settings.

2. Back-End Architecture (The Foundation)

  • Solid Core: Database designed for scalability to prevent future structural changes.
  • Security First: Uses "Fuzzy Location" (approximate coordinates) to protect landlord privacy, similar to Airbnb/Apple.
  • LINE Integration: Since web apps cannot send native push notifications, the platform will use LINE Messenger for real-time user alerts.

‚ú® 3. Key Innovation: "1-Click Airbnb Migration"

  • The Feature: We utilize web crawling/RPA to automatically import property details and photos from existing Airbnb listings.
  • The Benefit: Solves a major pain point for landlords with multiple units (20+). No manual data entry required‚Äîsignificantly increasing conversion rates.

üó∫Ô∏è Strategy & Business Direction

  • Company Name
    "SiamO" (Siam + O suffix). Innovative, catchy, technical sound.
  • Target Market

    Focus: Entire Home Rentals (TAM: 800M THB/year)

    Goal: Capture 50M THB revenue (6.25% market share) within 18 months.

  • Service Strategy

    Avoid in-house construction teams for now.

    Approach: Connect via API to Q-Chang/HomePro (2-5% commission).

    Quality Control: Focus on due diligence inspections in Chiang Mai, Hua Hin, and Ubon.

  • Revenue Model
    1. Subscription Fees (Landlords)
    2. Transaction Fees
    3. Service Commission (Referrals)
